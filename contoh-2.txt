Belajar dan Tutorial Git untuk Pemula sampai Mahir

Git merupakan salah satu distributed version control yang saat ini paling populer digunakan di kalangan programmer, dengan banyaknya dukungan dari web besar seperti Github, Bitbucket dan GitLab, membuat Git menjadi standard wajib untuk programmer ketika berkolaborasi membuat aplikasi.

Pada course ini, kita akan belajar Git dari dasar sampai mahir. Materinya dibuat terstruktur dengan baik, berisi teori dan praktek yang bisa diikuti oleh programmer pemula. Harapannya peserta bisa menguasai Git dengan baik dan mahir. Materi Git ini berisikan semua teori Git sekaligus praktek dari teori nya, sehingga peserta bisa mengikuti materi ini secara bertahap.
